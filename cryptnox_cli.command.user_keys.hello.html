cryptnox_cli.command.user_keys.hello.hello module

Module for handling user authentication using Windows Hello.

Warning: Windows hello functions are called in separate thread as calling the script as module will set the thread mode and windows also sets it on import

RuntimeError: Cannot change thread mode after it is set.
